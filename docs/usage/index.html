One column (Default)

Render one-column layout if the following elements exist body>main

<body>
  <main>
</body>

Two column (Optional)

Render two-column layout if the following elements exist body>(main+aside)

<body>
  <main>
  <aside>
</body>

About adjacent elements

Adjacent elements are not expected to affect the layout body>(any)+(main+aside)+(any) is permitted

<body>
  ...
  <main>
  <aside>
  ...
</body>

Top navbar (optional)

Render responsive top navbar if the following elements exist body>nav

<body>
  <nav>
  ...
  ...
</body>

Good practice

Don't place text elements directly under the body. Instead group them inside content sectioning elements.

NO: body>(p+p+p+p)

YES: body>(header>p)+(main>article>p)+(footer>p)

YES: body>(section>p)+(section>p)+(section>p)

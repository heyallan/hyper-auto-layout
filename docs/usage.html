One column (Default)

Render one-column layout if the following elements exist body>(any)

<body>
  <main>
</body>

Two column (Optional)

Render two-column layout if the following elements exist body>(main+aside)

<body>
  <main>
  <aside>
</body>

About adjacent elements

Adjacent elements are not expected to affect the layout body>(any)+(main+aside)+(any) 

<body>
  ...
  <main>
  <aside>
  ...
</body>

Note

For consistency always group inline and text elements under block sectioning elements. Do not place inline elements directly under body.

NO body>p+p+p

YES body>(header>p)+(main>p)+(footer>p)

YES body>(section>p)+(section>p)+(section>p)
